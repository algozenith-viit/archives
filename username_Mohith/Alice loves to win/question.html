Problem: Crazy XOR

Assuming you are friend of Alice. Alice always wanted beat Bob all time, but every time bob used to win. Now they have a challenge to find the number of unique paths in a grid such that the xor of path followed is 0. The path is followed from the cell (1,1) to (row,col) in one move you can move either one step right or one step down. i.e if you are currently at cell (x,y) you can move to either (x+1,y) or (x,y+1) iff (x < row and y < col). As alice is weak in these type of challenges and asked you help to win this round. Help Alice in this round to beat Bob.

Input Specification:

  • input1: The number of test cases t (1 <= t <= 103)
  • input2: The number of rows and number of columns (1 <= r,c <= 103).
  • input3: The value of the element at every cell (1 <= matrix[i][j] <= 103).
  • note r*c <= 1000

Output Specification:

Print the number of unique paths such that xor of those paths followed is 0.

Example

Input

        2

        3 2
        1 2
        1 2
        1 1

        3 3
        0 0 0
        0 0 0
        0 0 0

Output

        2
        6

Explanation

  • In test case 1
    The number of unique paths such that xor of those paths followed is 0 are 2.
    If you follow the path (1,1) -> (2,1) -> (3,1) -> (3,2) xor is 0.
    If you follow the path (1,1) -> (1,2) -> (2,2) -> (3,2) xor is 0.
  • In test case 2
    The number of unique paths with xor 0 are 6.
